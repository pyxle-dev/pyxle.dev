# Playground

Run Pyxle in your browser — no install. The interactive playground compiles and previews `.pyxl` files live, so you can see how Python `@server`/`@action` code and React components share one file. Open it at https://pyxle.dev/playground

To run Pyxle locally instead:

```bash
pip install pyxle-framework
pyxle init my-app
cd my-app && pyxle dev
```
